Create 3 new dashboard pages that follow the same UI style, colors, typography, and layout as the existing page design. The system is an Internship Management Platform for companies.

1. Create Internship Page

Design a page where the company can create a new internship.

Elements required:

Page title: Create Internship

Form fields:

Internship Title

Department

Internship Description

Required Skills

Duration

Location

Number of Positions

Application Deadline

Buttons:

Create Internship

Cancel

After submission, the internship must appear automatically in the Internships Page.

2. Internships Page

Design a page that displays all created internships.

Elements required:

Page title: Internships

Table listing internships with columns:

Internship Title

Department

Duration

Location

Status (Open / Closed)

Actions

Actions:

View

Edit

Delete

Assign Supervisor

3. Supervisor Management Page

Design a page where the company manages supervisors.

Sections required:

A. Supervisor Applications

Table showing supervisors who registered and are waiting for approval.

Columns:

Name

Email

Department

Application Date

Actions:

Accept

Reject

B. Accepted Supervisors

Second table showing approved supervisors.

Columns:

Name

Email

Department

Assign Internship Button

Action:

Assign Internship → company selects an internship to supervise.

4. Internship Certificate Page

Create a page to generate and manage internship certificates.

Elements:

Table showing:

Student Name

Internship Title

Supervisor

Duration

Status

Actions:

Generate Certificate

Download PDF

View Certificate

Certificate preview should include:

Company logo

Student name

Internship title

Supervisor name

Internship duration

Signature section

Design Requirements

Keep the same layout as the current dashboard

Use cards and clean tables

Responsive layout

Modern admin dashboard style

Consistent spacing, icons, and buttons
